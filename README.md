# Anflix Frontend

Static multi-page frontend for Anflix with unique layouts per section.

## Pages
- `index.html` (Home)
- `Pages/movies.html`
- `Pages/series.html`
- `Pages/trailers.html`
- `Pages/upcoming.html`
- `Pages/contact.html`

## Sections On Home
- Explore Sections (Movies, Series, Trailers)
- Upcoming Movies & Series (2 columns x 3 rows)
- Popular Movies
- Trending Movies Series


## Footer Links
- Contact and Advertise link to `Pages/contact.html`
- Privacy links to `privacy.pdf`
