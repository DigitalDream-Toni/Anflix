# Anflix Frontend

Static multi-page frontend for Anflix with unique layouts per section.

## Pages
- `index.html` (Home)
- `Pages/movies.html`
- `Pages/series.html`
- `Pages/trailers.html`
- `Pages/upcoming.html`
- `Pages/contact.html`

## Sections On Home
- Explore Sections (Movies, Series, Trailers)
- Upcoming Movies & Series (2 columns x 3 rows)
- Popular Movies
- Trending Movies Series

## Footer Links
- Contact and Advertise link to `Pages/contact.html`
- Privacy links to `privacy.pdf`

## Privacy PDF
- `privacy.pdf` is a multi-page, professionally formatted notice.

## Assets
- `assets/logo.svg`
- `assets/favicon.svg`

## Run Locally
Open `index.html` directly in your browser (no build step required).

## Note
Project README is kept up to date with changes.

## Search
The search box filters cards/tiles on the current page and highlights matches.
