# Anflix Frontend

Static multi-page frontend for Anflix with unique layouts per section.

## Pages
- `index.html` (Home)
- `movies.html`
- `series.html`
- `trailers.html`
- `music.html`

## Assets
- `assets/logo.svg`
- `assets/favicon.svg`

## Run Locally
Open any page directly in your browser (no build step required).

## Search
The search box filters cards/tiles on the current page and highlights matches.
